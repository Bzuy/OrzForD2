💻 Mã Giả


1. function binarySearch(arr, x) 
2.     left = 1, right = arr.length
3.     while (left ≤ right)
4.         mid = (left + right) / 2
5.         if arr[mid] == x then return mid
6.         else if arr[mid] < x then left = mid + 1
7.         else right = mid - 1
8.     return -1
